# color_aimbot
Simple AHK-like color aimbot script in python, as my first non-course project

## Requirements:
To run this program, you'll need to install the following libraries:
- keyboard
- numpy
- pyautogui
- pynput
- pywin32
- pillow
- psutil


You can install them by running the following command in your terminal:
```bash
pip install -r requirements.txt
```

## How to Run the Script:
Run aimbot.py silently using the following command:
```
pythonw.exe aimbot.py
```
